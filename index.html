MartinMade3D Assistant

MartinMade3D Assistant is an internal seller-support tool used only by our shop. It connects to Etsy via OAuth to read our own conversation threads and limited order metadata so we can:

  • Summarize message history
  • Identify the most recent buyer question
  • Draft a courteous, on-brand reply for a human to review and send

How it works

  • Authentication uses Etsy OAuth; access is restricted to our shop staff.
  • Data access is read-only for conversations and order metadata needed for support.
  • No automatic messaging or listing edits—humans review and send replies.
